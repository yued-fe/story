2017年8月25日

本周五，莫朵乐给我们分享了《起点海外版的经验之谈》，马头也给我们分享了《如何支持1000个运营活动》的技巧！会后大家表现出了对分享人的极大赞美！欲听从速！每双周五下午我们在千秋不见不散！

2017年8月18日

贪吃小分队出山啦！吃遍张江的小目标怕是要变成吃垮张江！这次我们吃的是上海大饭堂，由于物美价廉，每个菜都来它个三五...六七八份！结果！预期比大家想象得都要高，于是足智多谋的前端公社社长为大家想出一个一人免单的办法！学以致用，小程序走你!

2017年8月11日

慕名而来听咱前端公社双周会的小伙伴日渐增多！我们占领了阅文最大的会议室——青龙&玄武！喝着最火喜茶的前身——皇茶！听着最帅的人分享——最前沿的技术！

2017年8月6日

设计公社の香港自由行！陈冠中说香港是半唐番城市美学的代表，是伟大的杂种，是一半华、一半夷，一半中、一半外，一半人、一半鬼的混搭；从路边摊到米其林，从尖沙咀到油麻地，这里是美食的天堂也是购物的天堂，文化遗迹和历史遗迹遍布每个角落。能把香港的每一栋建筑，每一个角落都拍得如此美丽也非我们设计公社的小伙伴莫属了！
